================================================================================
API QUICK REFERENCE
================================================================================

CORE FUNCTIONS (utils/layer_access.py):

  get_layer_count(model) → int
    Get number of hidden layers from model.config

  validate_layer_idx(model, layer_idx) → bool
    Check if layer index is valid

  get_intermediate_hidden_states(model, input_ids, attention_mask, 
                                  layer_idx, requires_grad=False)
    → (hidden_states, attention_mask)
    Forward pass up to specified layer

  pool_hidden_states(hidden_states, attention_mask, pooling="mean")
    → pooled_features
    Average pooling over sequence dimension

  extract_single_layer_features(model, dataset, device, layer_idx, ...)
    → np.ndarray [N, hidden_size]
    Batch feature extraction

CLUSTERING CLASS (clustering/early_exit_kmeans.py):

  class EarlyExitKMeansClusterer(_KMeansBase):
    
    fit_with_intermediate_layer(dataset, model, tokenizer, device,
                                 cfg, layer_idx=-1, rank=0)
      → np.ndarray [N] cluster IDs
      Main entry point for intermediate layer clustering

================================================================================
USAGE EXAMPLES
================================================================================

BASIC USAGE:
-----------
from clustering import EarlyExitKMeansClusterer
from utils.layer_access import get_layer_count

clusterer = EarlyExitKMeansClusterer()
num_layers = get_layer_count(model)

# Cluster using middle layer
cluster_ids = clusterer.fit_with_intermediate_layer(
    dataset, model, tokenizer, device, cfg,
    layer_idx=num_layers // 2,  # Middle layer
    rank=0
)

EXPLORE LAYER-WISE FEATURES:
------------------------------
from utils.layer_access import extract_single_layer_features

for layer_idx in range(0, 36, 6):
    features = extract_single_layer_features(
        model, dataset, device, layer_idx, batch_size=32
    )
    print(f"Layer {layer_idx}: shape={features.shape}, "
          f"mean={features.mean():.4f}")

CONFIGURATION-BASED:

from utils.config import load_config

cfg = load_config('configs/default.yaml', overrides=[
    'clustering.early_exit.layer_idx=18',
])

cluster_ids = clusterer.fit_with_intermediate_layer(
    dataset, model, tokenizer, device, cfg, rank=0

================================================================================
TROUBLESHOOTING
================================================================================

Q: "Invalid layer_idx" error
A: Check layer_idx is in range [0, num_layers-1]
   Use: get_layer_count(model) to verify available layers

Q: "Model has no .config attribute"
A: Ensure model is HuggingFace CausalLM
   DDP wrapper still exposes .config through .module

Q: Out of Memory (OOM)
A: Try earlier layer (smaller hidden states)
   Or reduce feature_batch_size in config

Q: Slow feature extraction
A: Use earlier layer for faster extraction
   Ensure model is on GPU
   Increase batch_size if memory allows
